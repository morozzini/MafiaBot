# Mafia discord bot
Bot for help  in mafia game

## Config
Set your config in config.json  
**token** - The token of your bot - [get here](https://discordapp.com/developers/applications/me)  


## Need Install
* node.js
* discord.js `npm install discord.js`

## Run
```
node bot.js
```
